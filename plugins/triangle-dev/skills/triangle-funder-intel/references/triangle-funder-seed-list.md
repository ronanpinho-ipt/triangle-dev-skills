## Row schema

Each funder row carries these fields:

- **Name** — legal/common name of the funder
- **Type** — category (see sections below)
- **Priorities** — stated giving priorities / focus areas
- **Geography** — documented giving footprint (county/city/state)
- **Ask range** — documented typical grant range (from 990 or guidelines)
- **Deadline / window** — published application window, or "rolling," or none
- **Warm-path notes** — only real, documented connection routes
- **Source URL** — primary source the row was verified from
- **Last verified** — date of verification
